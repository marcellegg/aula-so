# 1. Período Pré-Sistema Operacional (1945–1955)

## 🖥 1945 — ENIAC  
**Desenvolvedor:** Exército dos EUA  

**Características:**
- Não possuía sistema operacional  
- Programação via cabos físicos e chaves  
- Execução manual  
- Um programa por vez  

**Impacto:**
- Início da computação eletrônica  

---

## 🖥 1951 — UNIVAC I  
**Desenvolvedor:** Remington Rand  

**Inovação:**
- Primeiro computador comercial  
- Uso empresarial e governamental  

---

# 2. Primeiros Sistemas Operacionais (1956–1968)

## 🖥 1956 — GM-NAA I/O  
**Desenvolvedor:** General Motors + IBM  

**Inovação:**
- Primeiro sistema operacional Batch  
- Execução automática de tarefas  

**Características Técnicas:**
- Processamento em lote  
- Uso de cartões perfurados  
- Sem interação do usuário  

---

## 🖥 1964 — OS/360  
**Desenvolvedor:** IBM  

**Inovação Principal:**
- Padronização de software para diferentes hardwares  
- Multiprogramação inicial  

**Impacto:**
- Marco na engenharia de software  

---

# 3. Consolidação do Modelo Unix (1969–1980)

## 🖥 1969 — UNIX  
**Desenvolvedor:** AT&T Bell Labs  

**Inovações:**
- Multiusuário  
- Multitarefa  
- Portabilidade  
- Sistema de arquivos hierárquico  

**Impacto:**
- Influenciou Linux, macOS e Android  

---

## 🖥 1973 — CP/M  
**Desenvolvedor:** Digital Research  

**Inovação:**
- Padrão para microcomputadores  

---

## 🖥 1977 — Apple DOS 3.1  
**Desenvolvedor:** Apple  

**Importância:**
- Popularização do computador pessoal  

---

# 4. Era dos Computadores Pessoais (1981–1999)

## 🖥 1981 — MS-DOS  
**Desenvolvedor:** Microsoft / IBM  

**Características:**
- Interface textual  
- Sistema de arquivos FAT  
- Monotarefa  

---

## 🖥 1984 — System 1 (Macintosh)  
**Desenvolvedor:** Apple  

**Inovação:**
- Popularização da Interface Gráfica (GUI)  

---

## 🖥 1985 — Windows 1.0  
**Desenvolvedor:** Microsoft  

**Inovação:**
- GUI sobre MS-DOS  

---

## 🖥 1991 — Linux (Kernel)  
**Desenvolvedor:** Linus Torvalds  

**Impacto:**
- Movimento open source  
- Base de servidores modernos  

---

## 🖥 1995 — Windows 95  
**Desenvolvedor:** Microsoft  

**Inovações:**
- Menu Iniciar  
- Plug & Play  

---

---

# 9. Conclusão Analítica

A evolução dos sistemas operacionais demonstra um padrão claro:

- Redução de tamanho  
- Aumento de desempenho  
- Maior interatividade  
- Mobilidade  
- Conectividade constante  
- Integração com IA  

O futuro aponta para sistemas autônomos, inteligentes e integrados à inteligência artificial de forma nativa.
